# Integration Map

## ChatGPT / Zy

Command layer, reasoning engine, audit layer, drafting support, and decision framing.

## Google Drive

Operational source-of-truth: dashboard, matter registry, ledgers, source tables, and live documents.

## GitHub

Version-control layer: canonical markdown files, changelog, issue tracker, templates, tests, and future automation scripts.

## Layer Rule

Drive is operational truth. GitHub is system/version truth. ChatGPT coordinates through available connectors and explicit Pram approval.
